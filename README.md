# Car Project Backend

## Overview
The **Car Project Backend** is designed to manage car listings, user authentication using JWT, and the ability to create, read, update, and delete car records. This project uses **Express.js** for handling API routes, **MongoDB** for storing data, and **JWT** for authentication.

## Features
- **User Authentication**: User sign-up, login, and JWT token generation for authentication.
- **Car Management**: Admins can add, update, and delete car listings.
- **Secure Routes**: All sensitive routes are protected using JWT.
- **Car Listing**: Display and manage car listings with detailed information.

## Technologies Used
- **Backend Framework**: Express.js
- **Database**: MongoDB (NoSQL database)
- **Authentication**: JWT (JSON Web Tokens)
- **Password Hashing**: bcryptjs
- **Environment Variables**: dotenv
- **CORS Handling**: cors
